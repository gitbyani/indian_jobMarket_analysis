# indian_jobMarket_analysis
Interactive Power BI dashboard analyzing India's job market, including salary trends, hiring patterns, work modes, location tiers, and industry insights.

This project presents an interactive Power BI dashboard designed to explore key trends in the Indian job market. The analysis focuses on salary distribution, hiring demand, work mode preferences, location-based opportunities, and industry performance. Rather than simply creating visualizations, the project emphasizes business-oriented insights that help understand where opportunities are concentrated, how compensation varies across experience levels, which industries offer higher salaries, and how companies are adapting to remote and hybrid work environments.

The dashboard was developed through multiple stages, including data cleaning, KPI selection, visual design, dashboard structuring, and insight generation. Special attention was given to creating a professional user experience, meaningful storytelling, and actionable insights that could be valuable for job seekers, recruiters, and business decision-makers. The final solution consists of a Home Page, Market Overview Dashboard, and Salary & Company Insights Dashboard, providing a comprehensive view of India's evolving employment landscape.

## Key Insights
- Tier 1 cities account for 42% of total job opportunities.
- Tier 2 cities contribute 40% of job opportunities.
- Remote jobs represent 18% of available openings.
- Lead and Senior professionals contribute 72% of salary distribution.
- Work mode analysis highlights the continued dominance of on-site roles.
- Industry-wise analysis identifies the highest-paying sectors in the market.
- Company-level analysis reveals organizations actively offering remote opportunities.
